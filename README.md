# Maldives Dream Stay Booking System

This is a **Java console application** that simulates a real-world hotel reservation process. It provides a structured way for users to explore resorts in the Maldives and manage their stay details through a terminal interface.

### **Application Features**
* **Hotel Discovery:** Browse hotels categorized by star ratings and view guest feedback.
* **Booking Management:** Collects guest information, room types, and preferred views.
* **Automated Pricing:** Calculates total costs including room rates, transfers, and optional activities.
* **Discount Logic:** Applies family discounts and promotional codes based on specific group conditions.
* **Payment Simulation:** Supports simulated card payments and bank transfers with receipt generation.
* **Input Validation:** Ensures data integrity for names, dates (**DD-MM-YYYY**), and payment details.

### **Technical Stack**
* **Language:** Java
* **Concepts:** Object-Oriented Programming (**OOP**), Data Validation, and Logical Flow.

### **How to Run**
1. Ensure you have the **Java Development Kit (JDK)** installed.
2. Download the source files.
3. Compile the program: `javac Main.java`
4. Run the application: `java Main`
